Search uses the official YouTube Data API, which needs a free API key tied to your own Google account — there's no way to bundle one that works for every visitor.

Get one free: console.cloud.google.com — new project → enable "YouTube Data API v3" → Credentials → Create API key. Restrict it to your domain when you're done testing. Free tier covers about 100 searches a day.

youtube — rules

  • Type a search into the box (or run youtube <query> from the main terminal) and hit [search] to see a grid of matching videos with thumbnails.
  • Click any result to watch it right here, embedded in the page — nothing opens in a new tab.
  • [\u2190 back to results] returns to your last search without re-searching.
  • Search needs a free YouTube Data API key from your own Google account, set once via [api key]. YouTube doesn't allow its own search page to be embedded on other sites, so this is the only way to get real results here — playback itself doesn't need a key.
  • The free tier covers roughly 100 searches a day; it resets at midnight Pacific time.
